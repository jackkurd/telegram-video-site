# پێشاندەری ڤیدیۆی تێلێگرام 🎬

وێبسایتێکی جوان بە ستایلی iOS بۆ پێشاندانی ڤیدیۆکانی تێلێگرام.

## چۆن دایبمەزرێنم لە Netlify؟

### هەنگاو 1: ئامادەکردنی فایلەکان
هەموو فایلەکانی ئەم فۆڵدەرە لە کۆمپیوتەرەکەت پاشەکەوت بکە.

### هەنگاو 2: بڵاوکردنەوە لە Netlify

#### ڕێگای GitHub (پێشنیارکراو):
1. فۆڵدەرەکە Upload بکە بۆ GitHub repository
2. بڕۆ بۆ [app.netlify.com](https://app.netlify.com)
3. کلیک بکە لەسەر "Add new site" > "Import an existing project"
4. GitHub هەڵبژێرە و repository-کەت دیاری بکە
5. "Deploy site" بکە

#### ڕێگای Netlify CLI:
```bash
npm install -g netlify-cli
cd telegram-video-site
netlify deploy --prod
```

### هەنگاو 3: دڵنیابوونەوە لە بۆتەکە
- بۆتەکەت پێویستە ئەندامی ئەو کەناڵانە بێت کە دەتەوێت ڤیدیۆکانیان ببینیت
- بۆتەکە پێویستە Admin بێت لە کەناڵە تایبەتەکاندا
- بۆ کەناڵە گشتیەکان، تەنها پێویستە ئەندام بێت

## چۆن بەکاری بهێنم؟

1. لینکی ڤیدیۆی تێلێگرام لە خانەکە دابنێ
2. Enter بگرە یان کلیک لە نموونەکان بکە
3. ڤیدیۆکە ڕاستەوخۆ نیشان دەدرێت!

- Admin بیکە (بۆ کەناڵە تایبەتەکان)
